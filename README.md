# Axel Alexius Latukolan — Portofolio

Selamat datang di repository portofolio profesional saya.

## Tentang
Repository ini berisi source code website portofolio digital milik Axel Alexius Latukolan, menampilkan pengalaman, keahlian, pengalaman kerja, serta berbagai informasi kontak dan pencapaian yang relevan di bidang hospitality dan teknologi.

Website ini dibuat dengan tampilan modern yang profesional, ramah mobile, dan fitur interaktif seperti animasi, parallax effect, theme mode (dark/light), serta perlindungan konten dasar.

---

## Fitur Utama
- **Responsive Design** — Tampilan optimal pada perangkat mobile dan desktop
- **Animated UI** — Efek parallax dan animasi modern
- **Mode Gelap/Terang** — Pengguna dapat beralih tema sesuai preferensi
- **Blokir Copy & Klik Kanan** — Pengamanan konten dari pencurian cepat
- **Bahasa Ganda** — Bahasa Indonesia dan Inggris
- **Navigasi Intuitif** — Navigasi smooth dan cepat antar section
- **Profil Karir & Skills** — Seksi pengalaman, keahlian teknis dan non teknis
- **Integrasi WhatsApp** — Tombol kontak cepat via WhatsApp

## Cara Menjalankan Secara Lokal
1. Clone repo ini:
   ```bash
   git clone https://github.com/unknownkz/portofolio.git
   ```
2. Buka `index.html` di browser modern

Tidak membutuhkan build step tambahan. Semua source berada di folder utama dan siap untuk diperiksa.
